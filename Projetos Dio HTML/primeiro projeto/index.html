Estrutura basica html

O HTML é a estrutura básica da navegação na internet,a organização desses códigos é o que permite a criação de um hipertexto, que, por sua vez, são os elementos que levam à criação de imagens, textos, vídeos, sons e outros elementos gráficos diversos. Ao visualizar a estrutura de um código, você vai perceber que há uma série de palavras que são delimitadas por esse símbolo: “<>”

Principais tags do html

A tag HTML dentro do <> e com fechamento : é a tag que delimita o documento html. Somente o que estiver dentro das duas tags será levado em consideração para a configuração do site

A tag HEAD: são as instruções e propriedades do cabeçalho da página, dentro dela vai a tag TITLE

que é o titulo da página do site, aparece na ba da página no topo do navegador.

A tag BODY: Essa tag é para compreender o espaço em que todo os conteudos gráficos e visual da pagina web será organizado, como imagens, botões, graficos, caixa de pesquisa

Essas são as tags basica de estrutura html

Tags

As tags HTML são elementos usados ​​para marcar e formatar os conteúdos: textual ou visual em documentos HTML. Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.

Atributos de tags

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc…

Formatação de texto HTML

Existem varia tag que é usada para formatação de textos no html, exemplo:

A tag i e usada aninhando a palavra que você queira colocar em Itálico no texto.

A tag strong e usada aninhando a palavra que voçê queira colocar em negrito no texto.

A tag u e usada aninhando a palavra que voçê queira colocar em sublinhar no texto.

Existem muitas tag para formatação de texto indico uma referencia para pesquisa de tags, indico a w3schools como um local para pesquisa sobre tags.

Lista em html

As listas no HTML são divididas em Lista Ordenada ol e lista não ordenada ul

A sintaxe para a criação de uma lista é a seguinte: abre a tag ol (para lista ordenada) ou ul (para lista não ordenada) e fecha a tag esolhida, dentro dela vai a tag li onde vai ser inserido o valor da lista

Navegação de páginas, Link em HTML

Outra tag importante no HTML e a tag a (tambem chamada de tag de link), com essa tag podemos fazer navegação para fora da nossa pagina, navegar entre as nossas pagina e navegar dentro da nossa propria pagina
